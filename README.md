# Claude MCP Toolbox

## Configuration

1. Copy `.env.template` to `.env`:
   ```
   cp .env.template .env
   ```

2. Add your API keys to the `.env` file:
   ```
   BRAVE_API_KEY=your_key_here
   GITHUB_PERSONAL_ACCESS_TOKEN=your_pat_here
   ```

The `.env` file is automatically ignored by git to keep your keys secure.
